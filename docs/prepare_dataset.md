

## NuScenes
Download nuScenes V1.0 full dataset data (lidarseg included) and CAN bus expansion data [HERE](https://www.nuscenes.org/download). Prepare nuscenes data as follows.


**Download CAN bus expansion**
```
# download 'can_bus.zip'
unzip can_bus.zip 
# move can_bus to data dir
```

**Prepare pretrained models.**
```shell
mkdir ckpts
cd ckpts 
wget https://download.pytorch.org/models/resnet50-19c8e357.pth
```

**Folder structure**

**Prepare nuScenes data**

*We generate custom annotation files from the dataset. Corridor annotations are generated by finding Maximum Empty Rectangles iteratively.*

```
python tools/data_converter/corridor_nuscenes_convertor.py nuscenes --root-path ./data/nuscenes --out-dir ./data/nuscenes --extra-tag corridor_nuscenes --version v1.0 --canbus ./data
```

Using the above code will generate `corridor_nuscenes_infos_temporal_{train,val,test}.pkl`.
Note: Generating the annotations could take quite a long time (2-3 days). It is recommended to directly download the annotations files from [google drive](https://drive.google.com/drive/folders/1Hnv1lQwf1lZrGbs9FX2sgJCsCKzosU5U?usp=drive_link).
